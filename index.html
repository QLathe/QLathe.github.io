About Me

Projects

JQZ Events


Technologies: C# - Asp.NET Core, HTML/CSS, Bootstrap, MySQL, jQuery, Microsoft Identity, Github, System.Net.Mail

The JQZ Events app has everything that one would expect from a calendar, allowing its users to check their calendar, add events, set reminders, and RSVP to event invites. JQZ Events goes further. Users can lookup and send event invitations to their friends using their email. JQZ Events has a unique feature that allows you to show what days you have free. The feature is implemented when creating an event, showing what days your friends are available, helping you pick the best day to create events.

Creating JQZ Events was a collaborative effort that I'm proud of. Every member of our team was instrumental in creating an easy to look at, user-friendly app. Creating a interactive and adaptive calendar, ensuring event's wouldn't overlap, and integrating email reminders into the app presented themselves as major challenges. Clear communication and teamwork were key to overcoming these challenges in a time sensitive environment.

Collaborators: John Albert, Zack Springer, Quinn Lathe

JQZ Dev Playground


Technologies: MERN Full-stack - MongoDB, Express, React, Node.js, HTML/CSS, Bootstrap, MaterialU

Developer Playground is a site created to showcase your digital personality. You can create custom portfolios, show off / view projects, practice solving algorithms, or spend some time playing small games. This site has a little bit of everything and anything that a developer could be looking for.

As a single page application, we had to pay special attention to each component and file structure to ensure that we wouldn't get in each other's way as we coded. We had a lot of fun creating this site, from implementing API's, to coding Tic-Tac-Toe. There were a lot of functions we were able to implement that helped take the project to the next level, which allowed it to feel like a fully functioning web site.

Collaborators: John Albert, Zack Springer, Quinn Lathe
